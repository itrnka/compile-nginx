# compile-nginx

Prepare script:
```bash
chmod +x install.sh
```

Display help:
```bash
sudo ./install.sh --help
```

Run script:
```bash
sudo ./install.sh --user my-nginx-user --group my-web-group --nginx-header-value my-server-name
```

Run script with default values:
```bash
sudo ./install.sh --default
```
